# 🎮 Xbox Subscription Dashboard (Dez/Jan)
## 📋 Descrição do Projeto
Este dashboard foi desenvolvido no Microsoft Excel para analisar o comportamento de renovação e o desempenho financeiro das assinaturas Xbox de Dezembro à Janeiro, de um ano fictício. O objetivo principal é segmentar os dados por status de renovação, tipo de contrato e volume de receita (tickets).
## 🧊Dados utilizados
Campos que foram utilizados da tabela: - Start Date, Auto Renewal, Subscription Type, Coupon Value.
## 📊 Oque foi observado no estudo:
- 1.Há mais inscritos em assinaturas mensais, com valor majoritário, em segundo há assinaturas trimestais.

- 2.Destes grupos de inscritos (mensais, trimestais, anuais), há os que optem por auto renovação, e há os que não queiram renovar.

- 3.O grupo que prefere renovar é consistente, mas há mais pessoas que não querem renovar. Principalmente os que assinaram mensal. Maioria majoritária dos assinantes anuais optaram por renovar ("Sim"), e apenas 2 "Não".

- 4.Novos assinantes também foram consistentes por meses, majoritariamente.

- 5.Os assinantes que não renovaram, tem mais valor de ticket, cerca de 30% a mais que o grupo de assinantes que renovaram.
